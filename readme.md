 # High Speed Shiritori Battle
## プロダクトの紹介 - 箇条書きにすると - 読みやすい
- 概要）しりとりしながら、技を出して、相手のバトルゲージを減らしてバトルする
- 使い方  
  1.ユーザー名を入力する  
  2.中央の文字にランダムで文字が出てる  
  3.しりとりスタート  
  4.しりとりしながら技を出す。技はしりとりしながら見つけていく
## 工夫した点，こだわった点
